Guia Básico de HTML e CSS

Resumo dos principais códigos e o que eles fazem

📌 Estrutura Básica

<!DOCTYPE html> → Define HTML5

<html> → Início do site

<head> → Configurações

<body> → Conteúdo visível

📝 Textos

<h1> até <h6> → Títulos

<p> → Parágrafo

<br> → Quebra de linha

<strong> → Negrito

🔗 Links e Imagens

<a href=""> → Link

<img src=""> → Imagem

📦 Containers

<div> → Bloco genérico

<section> → Seção

<header> → Cabeçalho

<footer> → Rodapé

📋 Listas

<ul> → Lista não ordenada

<ol> → Lista ordenada

<li> → Item da lista

🎨 CSS - Estilo

color → Cor do texto

background → Cor de fundo

font-size → Tamanho da fonte

margin → Espaço externo

padding → Espaço interno

border → Borda

📐 Layout

display: flex → Layout flexível

justify-content → Alinha horizontal

align-items → Alinha vertical
